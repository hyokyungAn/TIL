* Swiper라는 생성자 함수 실행
* 선택자 입력 ➡️ awards라는 클래스를 가지고 있는 section 요소 내부에 swiper-container라는 클래스를 가지고 있는 요소를 찾아서 슬라이드 기능 추가
* 필요한 옵션을 2번째 인수로 넣어라 ➡️ 객체 데이터 형태
* direction: horizontal; : 수평을 의미하는 문자 데이터, 이미 기본값으로 명시되어 있기 때문에 작성할 필요 없음
* slidesPerView: 한 번에 몇개의 슬라이드를 보여줄 것인가
